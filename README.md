## 動作環境
    Python+PyQtが使えれば動くハズ。

### OS

* Windows

    Windows7(32,64bitとも)でしかまともに動かしてません。

* Mac

    MountainLionで動かしたことあります。

* Linux

    動かしたことありません。

### Python

    PyScripterが3.4に対応してないので、3.3で開発してます。
    でも、たぶん3.4でも動くでしょう。

    こっから入手できます。
    http://www.python.jp/

### PyQt
    4.8.6で動かしてます。
    それ以上だったら動くんじゃない？

    バイナリはこっから入手できます。
    http://www.lfd.uci.edu/~gohlke/pythonlibs/


## 動かし方

    > git clone http://gitserver:port/git/wata/pmp.git
    > python pmp/pmp/main.py
