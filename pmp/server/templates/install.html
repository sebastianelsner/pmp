【起動方法】

    方法１：
      解凍したzipの中にある、pmp.pywをダブルクリックしてください。
      (拡張子.pyw が、Python3と結びつけられていなければ動きません）

    方法２：
      ・解凍したzipのフォルダで、コマンドプロンプトを開きます。
      ・コマンドプロンプトで、"python pmp.pyw"を実行してください。
